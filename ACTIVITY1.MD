Acividad 1: introduccion

En este proyecto, quiero que me ayudes a crear elementos esenciales para la aplicación EventEase, una aplicación de gestión de eventos que desarrollarás a lo largo de una serie de actividades.

Esta es la primera de tres actividades para crear la aplicación EventEase. El trabajo que completes aquí formará la base para las tareas de depuración y optimización en la Actividad 2 y la expansión en la Actividad 3.

Paso 1: Revisar el escenario

- Para empezar, revisa el siguiente escenario para comprender la intención de este proyecto.

- EventEase es una empresa ficticia especializada en la gestión de eventos corporativos y sociales. Le han encargado la creación de una aplicación web que permita a los usuarios:

- Navegar por los eventos con detalles como el nombre del evento, la fecha y la ubicación.

- Navegar sin problemas entre páginas como detalles del evento y registro.

- Como parte de este equipo de desarrollo, su responsabilidad es construir y perfeccionar el front-end utilizando Blazor.

Paso 2: Revisar los requerimientos

- Su primera tarea es crear el componente Event Card y configurar el enrutamiento básico para la aplicación EventEase. Estas características actuarán como base de la aplicación y permitirán futuras depuraciones y ampliaciones.

Paso 3: Generar un componente Blazor basico.

- Crea un nuevo componente Tarjeta de eventos.

- Hazme sugerencias para definir la estructura del componente e implementalas.

- Incluya campos para el nombre, la fecha y la ubicación del evento.

Paso 4: Implementar el enlazamiento de datos bidirecionales.

- Enlazamiento de datos para mostrar dinámicamente los detalles de los eventos.

- Utilice datos simulados o un modelo de datos sencillo.

- Sugiere y refine la sintaxis de enlace.

Paso 5: Configurar el enrutamiento entre paginas

- Por último, trabajarás en el enrutamiento entre páginas.

- Crea enlaces de navegación entre la lista de eventos, los detalles y las páginas de registro.

- Sugiere y verifica las rutas de enrutamiento, garantizando las mejores prácticas.

Paso 6: Guarde su trabajo

Al final de esta actividad, debes tener:

- Un componente Event Card funcional con campos y enlace de datos bidireccional.

- Funcionalidad de enrutamiento entre las páginas de la aplicación EventEase.

- Un código base, listo para ser depurado en la siguiente actividad.

- Asegúrate de que tu código se guarda en tu entorno sandbox.

- Los componentes y la funcionalidad que has construido aquí se utilizarán en la Actividad 2 para la depuración y optimización.

- Una vez que termines, prepárate para la siguiente actividad #2.
